# Example: "Wurzel-Flummery"

Wurzel-Flummery is a play by A. A. Milne, which was performed for the first time in 1917, in London.

This example is an edited version of the first few lines of the play, rendered in different versions:

- `full-cast/` contains the whole script readout,
- `no-viola/` is a version where the character of Viola is replaced by pauses,
- `richard-on-right/` is a version where Richard is on the right channel, while the rest of the cast is on the left.

The voices come from the "CSTR VCTK Corpus": see `voices/README.md` for more details including license information.
